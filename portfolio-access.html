Access

Levels Beyond had a legacy application written in Flex. I did a use case analysis of it, split certain functions into other applications, and redesigned the application to be written in HTML5 for presentation in desktop and tablet browsers. Where possible I tried not to use any desktop specific metaphors, with the exception of object selection in list and tile views. The tablet was always a first class citizen in the design and referees at Wimbledon used Access on iPads to review footage and make decisions.
