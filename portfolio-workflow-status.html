Workflow Status

The first part of this company's value proposition is easily finding and grouping the assets that a user is serching for. The second part is to easily run customized automated workflows on the set of assets. Examples include transcoding, uloading to endpoints via Aspera, archiving / restoring, etc. Some of these workflows can take hours to complete, so the user needs to be able to check on the status of the ones they or their team members have executed. I desiged this interface to replace the legacy application's screen after identifying the primary use cases and pain points with the previous application. This application is designed to quickly give an overview of the statuses or all relevant workflows and to immediately surface any errors with a single click. It works on both desktops and tablets.
